# #neoicecream GitHub Pages Site

Static one-page version prepared for GitHub Pages.

## Files
- `index.html` — main page
- `styles.css` — visual styling
- `script.js` — mobile menu behavior
- `assets/` — place real product/logo images here if needed

## Important
The contact form currently uses `mailto:info@neo-icecream.com` because GitHub Pages is static and does not process forms by itself. To collect form submissions properly, connect the form to Google Apps Script, Formspree, Netlify Forms, or another backend endpoint.

## Deploy
Upload these files to the root of the GitHub Pages repository and commit changes.
